# Ledger

## Installation

- Run `npm install` from root folder
- Create the ledger database

  ```
  export MY_APP_NAME=boilermaker
  createdb $MY_APP_NAME
  createdb $MY_APP_NAME-test
  ```

- You can create dummy accounts by running `node script/seed.js`
